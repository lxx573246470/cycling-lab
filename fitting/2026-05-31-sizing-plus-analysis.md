# 2026-05-31 Sizing Plus Fitting 解读

## 报告来源

- 原始文件：[刘潇枭-Sizing Plus.pdf](刘潇枭-Sizing%20Plus.pdf)
- Fitter：Fitfuel 运动表现 / Bikefitting 郭老师
- 车型：Sworks Tarmac SL5 52cm
- Fitting 日期：2026-05-31

## 关键身体评估

| 项目 | 报告结果 | 训练含义 |
| --- | --- | --- |
| 肱骨前引 | 肱骨前移 | 需要加强肩胛后缩、下压和胸椎伸展，减少长时间耸肩/圆肩 |
| 骨盆 | 轻度骨盆左旋 | 需要做单侧臀桥、死虫、鸟狗和髋稳定，减少左右发力差 |
| 膝轨迹 | 轻度膝内扣，胫骨外翻 | 需要加强臀中肌、外旋控制和单腿膝盖对齐 |
| 足部 | 轻度外八、轻度足内翻 | 骑行中关注锁片角度、足弓支撑和膝盖指向 |
| 腘绳肌 | 偏弱 | 需要髋主导后链训练，不只做拉伸 |
| 股四头肌/内收肌 | 左侧内收肌群紧张 | 需要内收肌活动度和左右对称控制 |
| 臀/梨状肌 | 双侧梨状肌紧张 | 需要臀肌激活、髋外旋活动度和避免久坐后直接高强度骑 |
| 髋关节 | 屈髋轻度受限 | 需要髋屈曲活动度、髋铰链和深蹲模式控制 |
| 踝关节 | 良好 | 暂不作为主要限制 |

## 车设变化

| 项目 | Fitting 前 | Fitting 后 | 适应重点 |
| --- | ---: | ---: | --- |
| 座垫高度 | 720 | 708 | 座高降低较多，先观察膝前侧和股四头肌负担 |
| 有效座垫后移量 | 205 | 190 | 坐垫更靠前，注意髋角度和膝盖轨迹 |
| 手变 Reach | 550 | 580 | 前伸增加，肩胛稳定更重要 |
| 手变 Stack | 605 | 620 | 前端更高，理论上有助于减轻腰背压力 |
| 把立 | 100 / -10° | 110 / 0° | 前端姿势变化明显，前 1-2 周避免长时间低把 |
| 上把宽度 | 380 | 400 | 肩宽匹配更好，但需避免耸肩 |
| 座垫角度 | 0° | -1° | 注意是否出现手部承重增加 |

## 优先加强动作

1. 髋主导后链：罗马尼亚硬拉、单腿臀桥、滑盘腿弯举或瑞士球腿弯举。
2. 臀中肌与膝轨迹：蚌式开合、侧向弹力带走、台阶下放、分腿蹲。
3. 骨盆抗旋转：死虫、鸟狗、Pallof press、侧桥。
4. 肩胛稳定：弹力带面拉、俯卧 Y-T-W、墙天使、胸椎伸展。
5. 髋活动度：90/90 转髋、髋屈肌拉伸、梨状肌拉伸、内收肌岩石式后坐。

## 骑行执行提醒

- 新车设前 2 周以适应为主，不做 FTP 测试和冲刺训练。
- 每次骑行前做 6-10 分钟髋和臀激活，尤其是户外起伏路或骑行台甜区课前。
- 骑行中每 10-15 分钟检查一次：肩放低、肘微弯、膝盖朝第二脚趾方向、踩踏从髋伸展开始。
- 如果出现单侧膝痛、髋外侧刺痛、手麻或肩颈紧张，记录出现时间、路况和握把位置，下一次训练降级。
