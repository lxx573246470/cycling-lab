# Plan Library

这里存放可复用的训练课模板。每周计划和每日安排应优先引用这里的模板，而不是复制完整内容。

## 分类

- `endurance/`: Z2、有氧基础、长距离耐力。
- `recovery/`: 恢复骑、休息日、低压力活动。
- `intervals/`: 甜区、阈值、VO2max、冲刺等间歇。
- `testing/`: FTP、心率阈值、基准测试和测试准备。
- `strength/`: 力量、灵活性、核心训练。

## 新增规则

1. 先搜索是否已有相近训练课。
2. 如果已有，周计划直接引用。
3. 如果没有，按分类新增一个 Markdown 模板，再在周计划中引用。
4. 如果该训练课需要导入 Zwift，再把 `.zwo` 文件放到 `workouts/zwo/`，并在模板中链接。
