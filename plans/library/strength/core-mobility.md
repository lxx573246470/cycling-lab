# 核心与灵活性

## 适用场景

- 非骑行日的辅助训练。
- 改善骑姿稳定性和髋部活动度。

## 训练结构

| 动作 | 组数 | 说明 |
| --- | --- | --- |
| 平板支撑 | 3 组 | 每组 30-60 秒 |
| 侧桥 | 2-3 组 | 每侧 20-45 秒 |
| 臀桥 | 3 组 | 每组 10-15 次 |
| 髋屈肌拉伸 | 2 组 | 每侧 45-60 秒 |
| 腘绳肌拉伸 | 2 组 | 每侧 45-60 秒 |

## 执行标准

- 不做到力竭。
- 不影响第二天骑行质量。
